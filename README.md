📈 Stock Portfolio Tracker - Python

📌 Project Overview

The Stock Portfolio Tracker is a Python-based application that helps users calculate the total value of their stock investments. Users can enter stock symbols and quantities, and the program calculates the investment value using predefined stock prices.

This project was developed as part of the CodeAlpha Python Programming Internship and demonstrates the practical use of Python for financial calculations and data management.


✨ Features

User-friendly stock input system

Calculates individual stock investment values

Computes total portfolio value

Uses predefined stock prices

Simple and interactive console-based interface

Option to save results in a text file
